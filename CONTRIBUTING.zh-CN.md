# 贡献指南

感谢您对 Awesome AI Waifu 项目的关注！我们欢迎所有形式的贡献。

## 如何贡献

### 添加新资源

1. Fork 本仓库
2. 创建您的特性分支 (`git checkout -b feature/AmazingResource`)
3. 在 README.md 的相应分类下添加新资源
4. 确保遵循以下格式：
   - `- [资源名称](链接) - 简短描述`
   - 描述应该简洁明了，不超过一行
   - 链接应该是直接可访问的
5. 提交您的更改 (`git commit -m 'Add some AmazingResource'`)
6. 推送到分支 (`git push origin feature/AmazingResource`)
7. 开启一个 Pull Request

### 资源添加指南

- **质量优先**：只添加高质量、有用的资源
- **避免重复**：检查资源是否已存在
- **分类正确**：将资源添加到最合适的分类
- **描述清晰**：提供简洁但信息丰富的描述
- **链接有效**：确保所有链接都是可访问的

### 分类说明

- **框架和工具**：开发框架、SDK、工具库等
- **模型和数据集**：预训练模型、数据集、模型权重等
- **应用和平台**：实际应用、在线平台、服务等
- **研究和论文**：学术论文、研究报告等
- **社区和资源**：论坛、Discord、Reddit、博客等
- **教程和指南**：教程、指南、文档等

### 提交信息规范

- 使用清晰、描述性的提交信息
- 如果是添加资源，使用格式：`Add: [资源名称]`
- 如果是修复，使用格式：`Fix: [问题描述]`
- 如果是更新，使用格式：`Update: [更新内容]`

## 行为准则

- 保持友好和尊重
- 欢迎不同观点和经验
- 优雅地接受建设性批评
- 关注对社区最有利的事情

## 问题反馈

如果您发现问题或有建议，请开启一个 Issue。

## 许可证

通过贡献，您同意您的贡献将在与项目相同的许可证下发布。
